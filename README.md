### Install the project 

1. Pull it 

2. Verify that nothing is running on port 80 : 


`npm install`
`npm run build`
`npm run work:server`

`docker-compose up`


**Note : The Docker command won't close it's output, but it is normal. Run : **

`docker ps`

And you should see *TWO* Containers : Node and MongoDB
